Here are Shiyun’s MRes miniproject files for CMEE.

There are 4 folders in this repository.

The Code folder contains the coding files used in this project.
1.Prepare_data.R which clear the dataset for analysis
2.Model_fitting.py which fit the models
3.Plot_graph.R which plot the data and carry on analysis
4.miniproject.tex and miniproject.bib are LaTeX files that form the project report
5.CompileLaTeX.sh is the bash code that compile the LaTeX code to pdf

The Data folder contains mata-data file.

The Result folder contains outputs from the coding files including project report in pdf.

The Sandbox folder contains some former version output files.
